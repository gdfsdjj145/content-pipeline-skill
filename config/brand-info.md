## HTML 模板占位符

技能会在生成 HTML 时替换以下占位符：

| 占位符 | 默认值 | 用法 |
|---|---|---|
| `{{BRAND_NAME}}` | 技术人自救宝典 | 公众号名称 |
| `{{BRAND_AUTHOR}}` | kajian | 作者名 |
| `{{CTA_TEXT}}` | 回复「2HB」获取 2-Hour Builder 方法论精华 | 公众号引流 |
| `{{ZHIHU_CTA}}` | 关注公众号「技术人自救宝典」，回复「2HB」获取完整方法论 | 知乎引流 |
| `{{XIAOHONGSHU_CTA}}` | 看我主页简介 | 小红书引流 |
| `{{BRAND_FOOTER}}` | 技术人自救宝典 · kajian | HTML 底部署名 |
